# MedCheck AI 🩺

An AI-powered medical symptom assessment platform that provides structured diagnostic reasoning, confidence scoring, and organ-system-based triage — built to help users understand their symptoms before seeing a doctor.

🔗 **Live Demo:** https://Shashwat045.github.io/medcheck-ai

## Features

- 🤖 Rule-based clinical reasoning pipeline with symptom clustering and risk stratification
- 🚩 Red-flag detection for urgent symptoms requiring immediate medical attention
- 📋 Patient-friendly explanations of complex medical reasoning
- 🔐 Secure login/signup authentication
- 📊 Medical history tracking dashboard
- 📄 PDF medical report export
- ☁️ Persistent cloud-based patient record storage

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** Supabase
- **AI/Logic:** Antigravity AI (rule-based reasoning engine)
- **Other:** REST APIs, PDF export

## How It Works

1. User inputs their symptoms through a guided interface
2. The reasoning engine clusters symptoms and cross-references organ-system-based triage logic
3. A confidence score and risk level are generated
4. Results are explained in plain language, with red-flag symptoms highlighted
5. Users can save their history and export a PDF report

## Running Locally

```bash
git clone https://github.com/Shashwat045/medcheck-ai.git
cd medcheck-ai
# Install dependencies
npm install
# Start the server
node server.js
```
